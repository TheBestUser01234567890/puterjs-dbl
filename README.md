## Installation
```
git clone https://github.com/TheBestUser01234567890/puterjs-dbl.git
cd puter.js
npm install
```

## Run development server

```
npm start
```

## Build

```
npm run build
```

### Example
Make sure the development server is running.

```html
<html>
<body>
    <script src="http://127.0.0.1:8080/dist/puter.dev.js"></script>
    <script>
        // Loading ...
        puter.print(`Loading...`);
    </script>
</body>
</html>
```
